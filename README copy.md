# Проект школы JS


## Установка проекта
`npm install`

## Запуск проекта
`npm start`


Фронт стартует на порту 3000
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

Бек стартует на порту 5002
Open [http://localhost:5002](http://localhost:5002) to view it in your browser.
